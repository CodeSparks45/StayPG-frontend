53. Maximum Subarray

Problem Statement: Given an integer array nums, find the subarray with the largest sum, and return its sum. Must solve in O(N) time.

KadanesAlgorithm.java
class Solution {
    public int maxSubArray(int[] nums) {
        int maxSum = nums[0];
        int currentSum = nums[0];

        // Kadane's Algorithm: O(N) Time, O(1) Space
        for (int i = 1; i < nums.length; i++) {
            currentSum = Math.max(nums[i], currentSum + nums[i]);
            maxSum = Math.max(maxSum, currentSum);
        }

        return maxSum;
    }
}

75. Sort Colors

Problem Statement: Given an array nums with n objects colored red (0), white (1), or blue (2), sort them in-place.
Optimal: One-pass algorithm using constant extra space.

DutchNationalFlag.java
class Solution {
    public void sortColors(int[] nums) {
        // Dutch National Flag Algorithm (O(N) Time, O(1) Space)
        int low = 0, mid = 0, high = nums.length - 1;

        while (mid <= high) {
            if (nums[mid] == 0) {
                swap(nums, low, mid);
                low++;
                mid++;
            } else if (nums[mid] == 1) {
                mid++;
            } else {
                swap(nums, mid, high);
                high--;
            }
        }
    }

    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}

121. Best Time to Buy and Sell Stock

Problem Statement: You want to maximize your profit by choosing a single day to buy one stock and choosing a different day in the future to sell that stock. Return the max profit.

Solution.java
class Solution {
    public int maxProfit(int[] prices) {
        // Track the minimum price seen so far
        int minPrice = Integer.MAX_VALUE;
        int maxProfit = 0;
        
        // O(N) Time, O(1) Space
        for (int i = 0; i < prices.length; i++) {
            if (prices[i] < minPrice) {
                minPrice = prices[i];
            } else if (prices[i] - minPrice > maxProfit) {
                maxProfit = prices[i] - minPrice;
            }
        }
        
        return maxProfit;
    }
}
